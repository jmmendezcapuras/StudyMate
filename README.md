# StudyMate

StudyMate is a multi-platform student study session logging application, made up
of a Spring Boot backend, a ReactJS web app, and (planned) an Android Kotlin
mobile app, all sharing a centralized PostgreSQL (Supabase) database via a
REST API.

- `backend/` — Spring Boot REST API (Java 17)
- `web/` — ReactJS + Vite web client
- `mobile/` — Android Kotlin app (planned)
- `docs/` — project documentation
